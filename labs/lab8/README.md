# vis-fix: Visual Debugger CLI

## Installation Dependencies

To install the core dependencies for this project, run:

```bash
npm install sharp openai @tavily/core
```

- **Pillow/sharp**: Used for image optimization and processing (sharp is the Node.js implementation).
- **openai**: Used for interacting with the vision-capable LLM.
- **tavily**: Used for fetching real-time documentation and solutions.
