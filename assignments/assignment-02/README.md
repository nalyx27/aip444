# Job Search Assistant

A comprehensive, AI-powered system designed to analyze the job market, evaluate your resume, and provide tailored application advice.

## Project Structure

- `analyze_market.ts`: Phase 1 CLI tool (Market Analysis).
- `analyze_resume.ts`: Phase 2 CLI tool (Resume Gap Analysis).
- `advise.ts`: Phase 3 CLI tool (Application Advisor).
- `src/`: Core logic and shared utilities.
- `raw_data/`: Input directory for your job postings and resume (ignored by git).
- `data/`: Structured JSON data generated by the system.
- `reports/`: Generated Markdown reports.

## Setup Instructions

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configuration**:
   Copy `.env.example` to `.env` and add your API keys:
   ```bash
   cp .env.example .env
   ```
   - `OPENROUTER_API_KEY`: Get from [OpenRouter](https://openrouter.ai/).
   - `TAVILY_API_KEY`: Get from [Tavily](https://tavily.com/).

3. **Prepare Data**:
   - Place at least 8 job posting PDFs or `.txt` files in `raw_data/jobs/`.
   - Place your resume (PDF or `.txt`) in `raw_data/resume/`.

## Usage

### Phase 1: Job Market Analysis
Processes all job postings, researches companies, and generates a market analysis report.
```bash
npx ts-node analyze_market.ts
```
Outputs:
- JSON data for each job in `data/jobs/`.
- `reports/market_analysis.md`.

### Phase 2: Resume Gap Analysis
Parses your resume and compares it against the market analysis to identify and triage gaps.
```bash
npx ts-node analyze_resume.ts
```
Outputs:
- `data/resume/resume.json`.
- `reports/gap_analysis.md`.

### Phase 3: Application Advisor
Analyzes a new job posting against your resume and market context to provide specific application advice.
```bash
npx ts-node advise.ts <path_to_new_job_pdf>
```
Outputs:
- `reports/application_report.md` (Fit assessment, resume tips, cover letter guidance, interview prep).

## Debug Mode
To see detailed logs including LLM calls, tool usage, and scoring breakdowns, set `LOG_LEVEL=debug` in your `.env` or prefix the command:
```bash
$env:LOG_LEVEL="debug"; npx ts-node analyze_market.ts
```
